## View Transitions

このサイトもview transisions に対応しており、大変良いユーザー体験となっています。

<OG url="https://developer.chrome.com/docs/web-platform/view-transitions/" />

Astroの場合はとても簡単で以下を行うだけで完成です。

```astro
---
import { ViewTransitions } from "astro:transitions";
---

<html lang="ja">
  <head>
    <title>私のホームページ</title>
    <ViewTransitions />
  </head>
  <body>
    <h1>私のウェブサイトへようこそ！</h1>
  </body>
</html>
```

<br />

<OG url="https://docs.astro.build/ja/guides/view-transitions/" />

しかしながら、view transisionsが有効になっているときにはページ遷移自体が通常の挙動と異なる点に注意が必要です。

## ナビゲーション

Astroの場合、以下の挙動となります。

- `<a>` をホバーすると、次のページのHTMLをprefetchする
  - `data-astro-reload` をつけると無効にできる
